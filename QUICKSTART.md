## Basic Controls

### Opacity Slider
- Drag to adjust transparency
- 0% = completely transparent
- 100% = fully opaque
- **Try 30-50% to see through the filter**

### Frequency Slider
- 0 Hz = static color (no alternation)
- 1-20 Hz = color alternation speed
- **Try 1-2 Hz for gentle effects**

### Color Pickers
- Click to choose your two colors
- Colors alternate based on frequency setting

### Preset Buttons
- Quick access to common frequencies
- Click any preset to instantly apply

## Tips for First-Time Users

1. **Start with low opacity** (30-40%) so you can see the effect
2. **Keep frequency at 0 Hz** initially to understand the basics
3. **Try different colors** - red/blue is just the default
4. **Drag the window** to position it where you want
5. **Use the hide button** (top-right) to get controls out of the way

## Common Use Cases

### Blue Light Filter (Night Mode)
1. Set opacity to 30-40%
2. Set frequency to 0 Hz (static)
3. Choose warm colors: Orange (#FF8800) and off

### Visual Effect
1. Set opacity to 20-30%
2. Set frequency to 1-2 Hz
3. Choose contrasting colors

### Screen Tint
1. Set opacity to 15-25%
2. Set frequency to 0 Hz
3. Choose a single subtle color
